# AiseoLogy

> **Analyze. Improve. Rank.**

A modern React-based SEO analyzer that evaluates HTML source code and detects common on-page SEO issues.

---

# 🇹🇷 Türkçe

## 📖 Proje Hakkında

AiseoLogy, HTML kaynak kodunu analiz ederek temel SEO kriterlerini değerlendiren React tabanlı bir web uygulamasıdır.

Uygulama; sayfa başlığı, meta açıklaması, başlık etiketleri, canonical URL, robots meta etiketi, Open Graph etiketleri, Twitter Card etiketleri, görsel alt (alt) metinleri ve boş bağlantılar gibi önemli SEO öğelerini analiz ederek kullanıcıya anlaşılır bir rapor sunar.

Bu proje, Front-End geliştirme becerilerini geliştirmek ve modern React mimarisini uygulamak amacıyla geliştirilmiştir.

# 🇬🇧 English

## 📖 About

AiseoLogy is a React-based web application that analyzes HTML source code and evaluates essential on-page SEO elements.

It inspects titles, meta descriptions, headings, canonical URLs, robots meta tags, Open Graph tags, Twitter Card tags, image alt attributes, and empty links, providing a simple SEO report with recommendations.

This project was developed to improve Front-End development skills while implementing a clean React architecture.

---

## ✨ Features

* HTML Source Code Analysis
* SEO Score (0–100)
* Title Detection
* Meta Description Detection
* H1 & H2 Analysis
* Canonical URL Detection
* Robots Meta Detection
* Open Graph Detection
* Twitter Card Detection
* Image Alt Analysis
* Empty Link Detection
* SEO Recommendations
* Responsive Interface
